How It Works
  1. Write your secret message
  2. Select a WAV audio file
  3. Choose the recipient
  4. System encrypts the message with recipient's RSA public key
  5. Encrypted data is hidden using LSB steganography
  6. Modified audio file is saved
Security

  • LSB steganography (undetectable)
  • SHA-256 file integrity
  • Full audit logging
